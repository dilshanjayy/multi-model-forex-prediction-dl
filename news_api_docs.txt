Basic pattern for EUR‑USD:

https://forexnewsapi.com/api/v1?currencypair=EUR-USD&items=100&page=1&date=MMDDYYYY-MMDDYYYY&token=YOUR_API_KEY

For example, for Q2 2021 (our first full quarter after history starts):

https://forexnewsapi.com/api/v1?currencypair=EUR-USD&items=100&page=1&date=04012021-06302021&token=YOUR_API_KEY

To cover 2021–Apr 2026, just iterate over 2–3 month chunks, e.g.:

- 04012021-06302021
- 07012021-09302021
- 10012021-12312021
- 01012022-03312022
- …continue same pattern…
- 04012026-04302026 (or today)

Inside each range, loop page=1…100 (stop when response has no items or `total_pages` is reached).

So in code, it’s basically:
1) list of date ranges
2) for each range, call that URL
3) paginate with `page` until done
